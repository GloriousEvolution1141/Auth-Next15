# App Dental

Aaplicativo web con **Next.js** y **Supabase**.

## Características
- Autenticación con cookies (`supabase-ssr`)
- Estilos con **Tailwind CSS** y **shadcn/ui**
- Funciona en cliente y servidor

## Desarrollo local
```bash
npx create-next-app --example with-supabase auth-supa-next
cd auth-supa-next
# editar variables de entorno
npm install
npm run dev
